# 電卓アプリ

VB.NET + WinFormで作成したシンプルな電卓アプリです。

## 機能

- 四則演算（+、-、×、÷）
- 小数点入力
- 符号反転（+/-）
- 1文字削除（delete）
- 連続計算（=を繰り返し押すと同じ演算を繰り返す）
- 計算履歴の保存・表示
- 履歴からの再利用（クリックで結果を入力値にセット）
- キーボード入力対応
- 入力桁数制限（10桁）

## 使い方

1. 数字ボタンまたはキーボードで数値を入力
2. 演算子ボタン（+、-、×、÷）を押す
3. 2つ目の数値を入力
4. =ボタンまたはEnterキーで計算実行

### 連続計算

=を繰り返し押すと、直前の演算が繰り返されます。

例: `5 + 3 =` → 8 → `=` → 11 → `=` → 14

### 履歴機能

- 計算結果は左側の履歴リストに保存されます
- 履歴をクリックすると、その結果を現在の入力値として使用できます

## キーマップ

| キー | 動作 |
|------|------|
| 0-9 | 数字入力 |
| + | 加算 |
| - | 減算 |
| * | 乗算 |
| / | 除算 |
| . | 小数点 |
| Enter, = | 計算実行 |
| Backspace | 1文字削除 |
| Escape, C | クリア |
