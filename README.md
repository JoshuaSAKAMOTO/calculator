# 電卓アプリ

VB.NET + WinFormで作成

## 機能

- 四則演算（+、-、×、÷）
- 小数点入力
- 符号反転（+/-）
- 1文字削除（delete）
- 連続計算（=を繰り返し押すと同じ演算を繰り返す）
- 計算履歴の保存・表示
- 履歴からの再利用（クリックで結果を入力値にセット）
- キーボード入力対応
- 入力桁数制限（10桁）

## キーマップ

| キー | 動作 |
|------|------|
| 0-9 | 数字入力 |
| + | 加算 |
| - | 減算 |
| * | 乗算 |
| / | 除算 |
| . | 小数点 |
| Enter, = | 計算実行 |
| Backspace | 1文字削除 |
| Escape, C | クリア |
